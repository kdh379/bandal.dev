이 블로그 포스트는 새로운 메타데이터 구조를 보여주기 위한 예시입니다. 이전에는 frontmatter 형식을 사용했지만, 이제는 Next.js와 더 호환되는 `export const meta` 구조를 사용합니다.

## 장점

1. Next.js와 더 나은 호환성
2. 자바스크립트/타입스크립트 객체 사용 가능
3. JSX를 메타데이터에 포함 가능

## 코드 예시

```javascript
const greeting = "안녕하세요!";
console.log(greeting);
```

위 코드는 간단한 인사말을 콘솔에 출력합니다.
